# プリンターサーバー（Electron版）

POS注文自動印刷システム

## 機能

- ✅ 自動印刷（新規注文を自動検出して印刷）
- ✅ GUI設定画面（簡単な設定）
- ✅ システムトレイ常駐
- ✅ 自動起動（PC起動時に自動的に起動）
- ✅ 接続テスト機能

## ビルド方法

### 1. 依存パッケージのインストール

```bash
npm install
```

### 2. Windows用EXEをビルド

```bash
npm run build:win
```

ビルドされたインストーラーは `dist` フォルダに生成されます。

## 開発モードで実行

```bash
npm start
```

## 必要な設定

- HerokuアプリURL
- 店舗ID
- APIキー（管理画面で生成）
- プリンタIPアドレス
- プリンタポート（通常は9100）
- チェック間隔（ミリ秒）

## アイコンについて

`assets/icon.png` と `assets/icon.ico` を配置してください。

アイコンがない場合は、デフォルトのElectronアイコンが使用されます。
